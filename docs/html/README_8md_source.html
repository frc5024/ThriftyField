1 # ThriftyField
2 ThriftyField is an open source alternative to the closed source FRC Field Management System (FMS). <br>
3 The goal of ThriftyField is to provide an easy way to manage the states of many robots at once just like the real fields to, but only with a small amount of inexpensive equipment. ThriftyField can also keep track of scores entered on tablets via the web application and can generate a near real-time audience view just like an official event.
4 
5 ## Main Features
6 These are all the main features of ThriftyField
7  - Web-based field control
8  - Manual scoring with tablets via the web application
9  - Near real-time audience display
10  - Chroma key display for using in a twitch stream
11  - Completely open API that can be used for building apps and other tools to interact with the field
12  - The ability to emulate an official offseason event network using the latest Driverstation protocol
13  - Only requires a laptop and any router to set up
14 
15 ## Setup
16 To set up ThriftyField, you will need the following:
17  - A laptop with python3 installed
18  - A wireless router with at least 7 ethernet ports, or a wireless router and a network switch
19  - At least 2 tablets or phones for score input
20  - (Optional) Other computer(s) connected to screens or projectors for displaying the audiance display
21 
22 ### The Network
23 Due to the way that the Driverstation protocol was designed, the field network must be set to the `10.0.100.x` ip space, with `10.0.100.5` reserved for the laptop running the ThriftyField software. After the router has been configured to accomodate this, either plug all 6 alliance station ethernet cables into the router, or a network switch.
24 
25 ### Scoring Devices
26 On the scoring devices, navigate to [10.0.100.5:8080](http://10.0.100.5:8080) in firefox or chrome and tap on the **Scoring** tab at the top, then select the alliance that the divice will be inputting data for.
27 
28 ### Audience Diaplay
29 On any computer hooked up to a projector or display, navigate to [10.0.100.5:8080](http://10.0.100.5:8080) in firefox or chrome, click on the **displays** tab, then select **Main**. Next, move this window to the secondary screen and press `f11` to go into fullscreen mode. Your mouse will be hidden when interacting with this window. If you are unable to hear game sounds, try enabling audio autoplay in your browser.
30 
31 ### Field Control
32 On the field control computer, navigate to [10.0.100.5:8080](http://10.0.100.5:8080) in firefox or chrome then click on the **Field Control** tab at the top.
33 
34 ## Usage
35 The following section outlines how to set up the field and score games
36 
37 ### Running the Software
38 On the field computer, make sure python3.6 or newer is installed, then install the required libraries with:
39 ```
40 pip3 install -r requirements.txt
41 ```
42 Next, run the program with:
43 ```
44 python3 main.py
45 ```
46 
47 ### Field
48 The field control panel is split into two sections: Alliance station control, and Game control.
49 
50 To add teams to the current match, simply type their team number into the corresponding text box to their assigned alliance station in the top section, then click anywhere outside of the text box to save. If you would like to start a game without a specific team or alliance station, simply click the green box beside the team number to bypass the alliance station. This box will turn red to show that the station has been disabled. Click it again to re-enable it.
51 
52 To start or abort a match, just click the corresponding buttons in the bottom section. ThriftyField will automatically handle the control data and networking for the robots and will follow the usual game timing and segments, along with switching between auto and teleop.
53 
54 ### Scoring
55 By default the scoring panels are set in addition mode. Pressing any number on screen will add it to the current score. For example, pressing 5 twice will add 10 to the score.
56 
57 To subtract from the score, press the **-** button to switch into subtraction mode. Make sure to press **+** when you want to start adding to the score again.
